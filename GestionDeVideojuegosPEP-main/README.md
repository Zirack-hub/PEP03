# GestionDeVideojuegosPEP
Repositorio creado para la pŕactica de Python de gestión de videojuegos.

Funcionamiento:
    El programa consiste en una lista de videojuegos (en utils_texto.py) que 
    se introduce en un diccionario llamado "catalogo" en main.py y con el 
    que se pueden realizar operaciones como: listar, buscar, filtrar, 
    añadir, actualizar y eliminar videojuegos. 
    Tambiense pueden mostrar estadísicas generales sobre estos.

El paquete src contiene:
    + busquedas.py (Contiene funciones para buscar o filtrar videojuegos)
    + catalogo.py (Contiene funciones CRUD)
    + estadisticas.py (Contiene la funcion que muestra conteo de todos 
                        géneros y numero por cada genero)
    + utils_texto.py (Contiene la lista de videojuegos y dos funciones 
                        para mostrar de manera clara y atractiva los videojuegos)
